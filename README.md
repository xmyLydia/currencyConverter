# Currency converter
Description: a site backed by a RESTful API that convert currency with value.
* Input
    * A source currency
    * A target currency
    * A monetary value
* Output
    * float monetary value of target currency
###The API leverages the exchange rates provided at https://exchangeratesapi.io
